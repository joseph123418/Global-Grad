# Global-Grad

Global-Grad is a responsive educational website designed to help students explore higher education opportunities, courses, scholarships, and study-abroad information. The platform provides a user-friendly interface that allows visitors to navigate through different sections and easily connect with the administrator for further assistance.

## Features

- Responsive and user-friendly design
- Multi-page website structure
- Course information section
- Scholarship information section
- Study abroad guidance
- Contact integration through WhatsApp
- Clean navigation between pages
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Project Highlights

- Developed a multi-page educational website from scratch.
- Implemented responsive design for desktop and mobile devices.
- Integrated WhatsApp-based contact functionality.
- Managed source code and version control using GitHub.
- Performed continuous updates and improvements through multiple commits.
